Download nodejs and npm from this website:
https://nodejs.org/en/

Download MongoDB Community Edition for installing database on your local computer:
https://www.mongodb.com/try/download/community

To install GUI for mongodb, download mongodb compass:
https://www.mongodb.com/products/compass

Create the new database named "lancer" in the mongodb compass

There are two directory to run commands in this project
The root folder : "/"
Frontend folder : "/Frontend/"

There are some package to install before running the project:
run "npm install" on both directories

Then, run "npm run server" on root folder

Run "npm run start" on Frontend folder. Running this command will launch the project on the browser.
If there are any port problems, edit the  number of port in .env file in both directories

Go to http://localhost:3000/login 
The project will start there
